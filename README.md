# NGA 用户画像分析器 — 安装与使用说明

## 概述

一款 Microsoft Edge / Chrome 浏览器扩展，用于分析 **nga.178.com** 论坛用户的公开历史发言，调用 AI API 生成多维度用户画像。分析维度包括：政治光谱、游戏偏好、性格语言风格、异常行为检测。

## 安装方法

### 方法一：开发者模式加载（推荐）

1. 打开 Edge 浏览器，地址栏输入 `edge://extensions/` 并回车
   - Chrome 用户输入 `chrome://extensions/`
2. 打开「**开发人员模式**」（Developer mode）开关
3. 点击「**加载解压缩的扩展**」（Load unpacked）
4. 选择本项目的 `nga-analyzer` 文件夹
5. 扩展加载成功后，工具栏会出现扩展图标

### 方法二：打包安装

1. 在 `edge://extensions/` 页面点击「**打包扩展**」
2. 选择 `nga-analyzer` 文件夹路径
3. 生成 `.crx` 文件后，拖入扩展管理页面安装

## 使用步骤

### 1. 配置 API Key

- 点击扩展工具栏图标 → 弹出窗口中点击「⚙️ 设置 API Key」
- 或右键扩展图标 →「扩展选项」
- 在设置页面输入你的 **API Key**
- 点击「保存」

### 2. 获取目标用户的发帖页面 URL

- 在 NGA 论坛中找到目标用户，点击其用户名进入用户详情页
- 在用户详情页点击「**发表的帖子**」标签
- 复制浏览器地址栏中的完整 URL，格式为：
  ```
  https://nga.178.com/thread.php?searchpost=1&authorid=xxxxxxxx
  ```

### 3. 开始分析

- 点击扩展工具栏图标打开弹窗
- 将目标 URL 粘贴到输入框中（如果剪贴板中有符合格式的 URL，会自动填入）
- 点击「**🔍 开始分析**」按钮
- 扩展自动抓取该用户最近 3 页公开发帖
- 将发言文本发送至 DeepSeek API 进行分析
- 分析完成后自动打开新标签页展示画像结果

### 4. 查看结果

结果页面包含：

| 模块 | 内容 |
|------|------|
| 🏛️ 政治光谱 | 倾向标签、可信度、简短描述 |
| 🎮 游戏偏好 | 游戏类型、涉及游戏、高频话题、总结 |
| 🧠 性格风格 | 性格关键词、语言风格、发言规律 |
| 🚨 异常行为 | 反串/引战等检测、典型发言片段 |
| 📝 整体总结 | 用户画像综合总结 |

### 5. 其他操作

- **刷新分析**：在结果页点击「强制刷新」跳过缓存重新分析
- **复制/打印**：支持一键复制纯文本结果、打印或保存为 PDF
- **清除缓存**：可在弹窗底部或设置页面清除所有已缓存的画像数据

## 技术说明

- **Manifest V3** 规范
- 支持的 NGA 域名：`nga.178.com`、`ngabbs.com`、`bbs.nga.cn`
- 请求权限仅限 NGA 域名范围
- 所有数据在本地浏览器处理，仅向 DeepSeek API 发送发言文本
- 分析结果缓存 7 天，存储在浏览器本地
- API 请求超时 15 秒，超时/失败可重试
- 同一用户 3 秒冷却，防止重复触发

## 触发方式

- **Popup 弹窗**：点击扩展图标 → 输入用户发帖页面 URL → 开始分析
- 支持 URL 格式：`https://nga.178.com/thread.php?searchpost=1&authorid=xxxxxxx`

## 异常处理

| 场景 | 处理方式 |
|------|----------|
| 未设置 API Key | 弹窗提示并引导前往设置 |
| 遭遇反爬/验证码 | 提示用户在浏览器中手动完成验证后重试 |
| API 超时/错误 | 显示具体错误信息，可重试 |
| URL 格式不正确 | 弹窗提示格式要求 |
| 用户无公开发帖 | 提示「未找到公开发帖记录」 |
| 缓存命中 | 直接展示缓存结果，可强制刷新 |

## 目录结构

```
nga-analyzer/
├── manifest.json          # MV3 扩展清单
├── popup.html             # 扩展弹窗（操作入口）
├── options.html           # 设置页面
├── result.html            # 结果展示页面
├── icons/                 # 扩展图标
│   ├── icon16.png
│   ├── icon48.png
│   └── icon128.png
└── lib/
    ├── service-worker.js  # 后台 Service Worker
    ├── popup.js           # 弹窗逻辑
    ├── options.js         # 设置页逻辑
    ├── result.js          # 结果页逻辑与渲染
    ├── result.css         # 结果页样式
    ├── api.js             # DeepSeek API 调用模块
    └── cache.js           # 本地缓存管理（7天过期）
```

## 隐私声明

- 扩展不上传任何数据至开发者服务器
- 唯一的远程通信是直接向 DeepSeek API 发送发言文本
- 分析结果仅缓存在浏览器本地存储中
- 仅分析无需登录即可访问的 NGA 公开内容
